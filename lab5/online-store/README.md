# Online Store

This Angular project was developed as part of the Web Development course (Lab 5).

The application demonstrates component architecture, category-based product hierarchy, and user interaction features such as liking, deleting, and sharing products.

---

## Features

- 4 product categories
- 20 real products from kaspi.kz (5 per category)
- Category-based filtering
- Like button with live counter update
- Delete button (removes product dynamically)
- WhatsApp and Telegram share functionality
- Responsive layout using CSS Grid / Flexbox
- Parent–child communication using @Input() and @Output()

---

## Technologies

- Angular (Standalone Components)
- TypeScript
- CSS
- Angular Control Flow (@for / @if)

## How to Run

1. Install dependencies:
npm install

2. Start the development server:
ng serve

3. Open your browser and navigate to:

http://localhost:4200/


---

## Project Structure

- AppComponent – category selection and state management
- ProductListComponent – displays products for selected category
- ProductItemComponent – individual product card with Like, Delete, and Share actions
- ProductService – stores categories and products data
- models/ – TypeScript interfaces
