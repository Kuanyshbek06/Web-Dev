The CSS border property defines a border around an HTML element. p { border: 2px solid powderblue; } The CSS padding property defines a padding (space) between the text and the border. p { border: 2px solid powderblue; padding: 30px; } The CSS margin property defines a margin (space) outside the border. p { border: 2px solid powderblue; margin: 50px; }
